# VRC6 - A small repository of text

##  Introduction

This is going to be a place where I will start archiving many of my thoughts, poetry, writings into one consolidated place. This site was created using Jekyll and Github Pages that provides me free hosting as long as there isn't any heavy binaries stored. This site also features a hidden AES encryption feature so I can share pages with a secret key. I went for a dark layout with a green text as it's easier on the eyes than having a bright white background. For now, the site seems barren but that's only because I haven't fully started archiving my writings as of yet but that will change soon enough. Scroll to the bottom to see the list of posts, thank you and may you find any of my work something worthwhile reading.

##  Recommended viewing

If you're seeing this through a computer browser, I would suggest to view with the widest possible window size. The site is made with a responsive design template meaning that the text for some poetry or writings can be re-aligned when resizing the window to be smaller, which may take away from its intentional purpose.

If you're viewing from your phone I would suggest viewing posts in widescreen mode for the same reasons I've mentioned previously and use the maximize link on top to fully extend the width margins.
